# 斗破插件架构

斗破插件按 `xiuxian_game` 的分层方式收敛职责，但保持 MVP 体量：

- `plugin.py`: 插件入口，只做懒加载。
- `plugin_web_routes.py`: FastAPI 路由，只负责页面、鉴权、参数转发和错误转换。
- `api_models.py`: Web API 的 Pydantic 入参模型。
- `core/`: 题材常量、境界表、默认行动、玩法模块说明。
- `features/`: Mini App bundle 组装层，负责把 SQL 服务、底栏、玩法模块汇总给前端。
- `shared/`: Telegram init data 与后台鉴权辅助。
- `static/`: 用户端和后台静态资源。
- `migrations/`: 插件自有数据库迁移。
- `bot/sql_helper/sql_doupo/`: 数据表模型和事务性服务层。

当前已开放玩法：

- 境界修炼：斗气、星级、境界推进。
- 魔兽历练：行动收益中产出金币和魔核。
- 炼药师：炼药经验、炼药师品阶和丹药库存。
- 丹药系统：炼药产丹，服丹提升斗气。
- 异火收服：异火线索推进，达到阈值后收服异火。
- 货币互通：Emby 碎片与斗破金币同事务兑换。
- 宗门势力：宗门贡献、职位和贡献消耗。
- 斗技功法：贡献研习斗技，斗技等级增加战力。
- 拍卖场：金币竞价获取丹药、魔核和异火线索。
- 魔兽 Boss：挑战高阶魔兽，获取魔核和战绩。
- 迦南塔修：塔层推进，高收益修炼。
- 斗气大陆游历：区域门槛、连续随机事件、风险路线、体力和撤离结算；前端支持路线滑动/方向键切换与蓄力聚气，聚气评级只提供有限战斗修正。
